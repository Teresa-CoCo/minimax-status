# 更新日志

## [1.1.0] - 2026-01-27

### 新增功能

- 新增 Token 消耗统计功能
  - 显示昨日消耗、近7天消耗、套餐总消耗
  - 支持万/亿单位自动格式化
- 新增账单记录 API 集成
- 账单数据 30 秒缓存，优化性能
